# Retail Sales Analysis

An end-to-end retail sales analysis project using **Python, SQL, and Power BI**.

## Technologies

- Python (Pandas)
- SQL (SQLite)
- Power BI
- DAX

## Project Structure

- **data_cleaning/** – Python notebook and script used to clean the dataset.
- **sql_queries/** – SQL scripts for exploration and business analysis.
- **dashboards/** – Power BI dashboard (.pbix) and PDF version.
- **dashboard_screenshots/** – Dashboard preview images.

## Analysis Includes

- Data Cleaning
- Data Exploration
- KPI Analysis
- Sales Analysis
- Product Analysis
- Interactive Dashboard

## Findings

The dashboard shows that the company generated €10.63 million in total revenue from 19,959 orders, with 5.57 million units sold. The average order value was €532.64, showing that customers spent a good amount per order.

The United Kingdom generated the highest revenue by a large margin. Sales increased towards the end of the year and reached their highest point in November. A few products generated most of the revenue, while some products sold many units but did not generate as much revenue. The dashboard also shows that some customers placed many more orders than others, making them valuable customers for the business.
