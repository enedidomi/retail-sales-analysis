# Retail Sales Analysis

An end-to-end retail sales analysis project using **Python, SQL, and Power BI**.

## Technologies

- Python (Pandas)
- SQL (SQLite)
- Power BI
- DAX

## Project Structure

- **data_cleaning/** – Python notebook and script used to clean the dataset.
- **sql_queries/** – SQL scripts for exploration and business analysis.
- **dashboards/** – Power BI dashboard (.pbix) and PDF version.
- **dashboard_screenshots/** – Dashboard preview images.

## Analysis Includes

- Data Cleaning
- Data Exploration
- KPI Analysis
- Sales Analysis
- Product Analysis
- Interactive Dashboard
